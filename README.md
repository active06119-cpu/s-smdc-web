# S-SMDC Landing Page

A sophisticated dark-mode landing page for S-SMDC (Surplus Energy-Based Small Modular Data Center) startup built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Modern Dark Theme**: Deep Space Blue and Dark Slate backgrounds with Emerald Green and Electric Blue accents
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Smooth Animations**: Scroll-triggered animations using Framer Motion
- **Component-Based Architecture**: Modular React components for easy maintenance

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React (Icons)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
